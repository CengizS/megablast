The MegaBlast

Aim

Shoot as many enemies as possible. Only projectiles will harm the ship, collisions with enemies do not matter
Currently this game is set up for endless mode - shoot 'em all!

Controls

Key Action
Arrow Keys Move ship
Spacebar Shoot
T Switch Track (one forward till it starts from the beginning)
C Radial shots (fires a bunch of projectiles, future "improvement" according to players abilities)
M Toggle Music on/off
